# C-Pong
Pong like game written in C with SDL2 library.

## Showcase Video
[https://www.youtube.com/watch?v=abHN47VsVEc](https://www.youtube.com/watch?v=abHN47VsVEc&ab_channel=Berkay%C3%87ubuk)

## Running the project

### Build
To build the game:
`make build`

### Run
To run the game:
`make run` or `./pong`
